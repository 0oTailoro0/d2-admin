# 导出数据

## 使用

### 注册插件

```
import pluginExport from '@/plugin/export'
Vue.use(pluginExport)
```

### 导出 `CSV`

使用方法

```
this.$export.csv({
  columns: this.table.columns,
  data: this.table.data
})
```

参数

| 参数名 | 介绍 | 必选 | 值类型 | 可选值 | 默认值 |
| --- | --- | --- | --- | --- | --- |
| columns | 列 | 非 | Array |  | 空数组 |
| data | 行数据 | 非 | Array |  | 空数组 |
